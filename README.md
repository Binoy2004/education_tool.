# EduRecon 🔍

### Educational Reconnaissance Toolkit (Bash)

EduRecon is a **lightweight reconnaissance toolkit written in Bash** designed for **cybersecurity students and ethical hacking learners**.
It demonstrates how basic **information gathering techniques** work during the **reconnaissance phase of penetration testing**.

The project focuses on **simplicity, education, and transparency**, helping beginners understand how common networking utilities operate in security workflows.

---

# ✨ Features

* ⚡ Interactive **CLI menu**
* 🎬 **Terminal animations**
* 🌐 **Ping network testing**
* 🔎 **DNS lookup**
* 📡 **WHOIS domain intelligence**
* 🧠 Beginner-friendly **Bash scripting**
* 📚 Designed for **cybersecurity learning environments**

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/edurecon.git
```

Navigate into the project directory:

```bash
cd edurecon
```

Make the script executable:

```bash
chmod +x EduRecon.sh
```

Run the tool:

```bash
./EduRecon.sh
```

# 🧠 How It Works

EduRecon automates simple **reconnaissance commands** commonly used during the early stage of penetration testing.

It performs:

1. **Connectivity testing** using ICMP ping
2. **DNS information gathering**
3. **WHOIS domain registration lookup**
4. **Terminal-based automation using Bash**

This allows students to learn:

* Bash scripting
* Network diagnostics
* Reconnaissance methodology
* Security tool automation
